# late july, midsummer wiki map
this project generates an interactive map of the late july, midsummer universe based on the wiki. It is written in golang and outputs an html file.
## usage
1. [download go](https://go.dev/dl/)
2. compile: `go build`
3. run binary `./ljms-wiki-map` or ljms-wiki-map.exe`

you can specify a center with the `-center` flag (default is the nexus), and a wiki url with the `-wiki` flag (default is https://latejulymidsummer.fandom.com/wiki/%s)
